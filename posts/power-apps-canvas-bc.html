¿Qué vamos a construir?

En este tutorial vamos a crear una Power App Canvas que se conecte a Business Central para mostrar una lista de clientes y permitir editar sus datos desde un dispositivo móvil.

Paso 1: Configurar el conector de BC

  1. Abre Power Apps (make.powerapps.com)
  2. Crea una nueva Canvas App en formato tablet o phone
  3. Ve a Data > Add data
  4. Busca Dynamics 365 Business Central
  5. Selecciona tu environment y company
  6. Elige las tablas/APIs que necesitas (ej: customers)

Paso 2: Crear la pantalla de lista

Añade una Gallery vertical y configura su Items:

SortByColumns(
    Filter(
        bcCustomers,
        StartsWith(displayName, SearchBox.Text)
    ),
    "displayName",
    SortOrder.Ascending
)

Dentro de la Gallery, añade labels para mostrar:

  • ThisItem.displayName - Nombre del cliente
  • ThisItem.number - Código de cliente
  • ThisItem.email - Email
  • ThisItem.phoneNumber - Teléfono

Paso 3: Pantalla de detalle

Crea una segunda pantalla con un Edit Form conectado al conector de BC. Configura:

  • Item: Gallery.Selected
  • DataSource: tu conector de BC (customers)
  • Campos a mostrar: displayName, email, phoneNumber, address

Para guardar cambios, en el botón de "Guardar":

SubmitForm(EditForm);
Navigate(ListScreen, ScreenTransition.None);
Notify("Cliente actualizado correctamente", NotificationType.Success)

Paso 4: Usar APIs custom

Si necesitas datos que no están en las APIs estándar, puedes crear una API page en AL:

page 50100 "Custom Items API"
{
    PageType = API;
    APIGroup = 'myGroup';
    APIPublisher = 'myPublisher';
    APIVersion = 'v1.0';
    EntityName = 'customItem';
    EntitySetName = 'customItems';
    SourceTable = Item;
    DelayedInsert = true;

    layout
    {
        area(Content)
        {
            repeater(General)
            {
                field(no; "No.") { }
                field(description; Description) { }
                field(unitPrice; "Unit Price") { }
            }
        }
    }
}

Consejos

  • Delegation: Las funciones como Filter y Sort se delegan al servidor si usas el conector nativo
  • Offline: Power Apps soporta modo offline con SaveData y LoadData
  • Performance: Selecciona solo los campos que necesitas con ShowColumns
  • Seguridad: Los permisos del usuario en BC aplican también en Power Apps
  • Publicación: Publica la app y compártela con tu equipo desde el admin center
